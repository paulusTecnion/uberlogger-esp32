Note: 16-bits will not improve resolution for temperature sensors, as these channels will be scaled down to 12-bits.

Averaging

Click to see the explanation

For sample frequencies below 1 Hz, you can choose to average the samples over the desired sample frequency or only take one sample at the desired sample rate. For example, if you enable this option with a sample frequency of 1 Sample/minute, the average will be taken over 1 minute of samples using an IIR filter. After 1 minute, the average will be output to the file.

Measurement Mode

Click to see the explanation

Three measurement modes are possible:

  • Continuous measurement: Instantly start the measurement using the "Start logging" button on the web interface or the physical button on the logger.
  • External trigger: Use the "Start logging" button on the web interface or the physical button on the logger to arm the logger. Then, wait for an external trigger using one of the digital inputs.
  • External control: Logger start and stop is controlled by a digital input. The logger start/stop button and web button are disabled.

Debounce time: Set the time for the trigger input to stabilize or delay the measurement. Maximum debounce time is 60000 ms (60 seconds).

Note: Currently, only triggers and external control on input HIGH are supported.

Note 2: The live view is not updating when in external trigger mode and the trigger is low.
